# APR oracles (aka APY oracles)

APR oracle allows you to get APR for various Liquid Staking tokens.

To call the oracle, choose a token and follow a link accordingly:
* [APR oracle for ankrBNB](/staking-for-developers/smart-contract-api/bnb-api/#get-apr)
* [APR oracle for ankrFTM](/staking-for-developers/smart-contract-api/ftm-api/#get-apr)
* [APR oracle for ankrPOL](/staking-for-developers/smart-contract-api/pol-api/#get-apr)
