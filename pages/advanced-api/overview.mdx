# Advanced APIs Overview

<Callout>
Advanced APIs is a unique feature that comes along with other extensive capabilities provided to our [Premium Plan](/rpc-service/service-plans/#service-plans-comparison) users.
</Callout>

> Start using Advanced APIs on [RPC Service](https://www.ankr.com/rpc/advanced-api/).

*Advanced APIs* is a specifically-tailored collection of JSON-RPC API endpoints built to support the most popular Web3 scenarios on multiple chains at almost instant speeds. We believe it to become a single point of reference for the multi-chain requests.
<br/>

<img src="/docs/build/API.png" alt="API" class="responsive-pic" />

Advanced APIs optimize, index, cache, and store blockchain data to make it access-ready for you. To make things even better, our Advanced APIs support querying eight EVM-compatible blockchains in a single request.

## Chains Supported

Currently, Advanced APIs work with the following EVM-compatible chains:

**Mainnets**:

  * `Ethereum`, `Fantom`, `BNB Smart Chain`, `Polygon`, `Polygon zkEVM`, `Avalanche`, `Arbitrum`, `Syscoin`, `Optimism`

**Testnets**:

  * `Ethereum Goerli`, `Polygon Mumbai`, `Avalanche Fuji`

And more EVM and non-EVM chains coming soon.

## API Methods Supported

Query our Advanced APIs at the following URL:

> `https://rpc.ankr.com/multichain`

### NFT API

[_NFT API Methods_](https://ankr.readme.io/reference/introduction#nft-api) serve to request NFT-related data across multiple chains:

  * [`ankr_getNFTsByOwner`](https://ankr.readme.io/reference/post_ankr-getnftsbyowner) — retrieves an account-associated NFTs.
  * [`ankr_getNFTMetadata`](https://ankr.readme.io/reference/post_ankr-getnftmetadata) — retrieves metadata of a particular NFT.
  * [`ankr_getNFTHolders`](https://ankr.readme.io/reference/post_ankr-getnftholders) — retrieves holders of a particular NFT.
  * [`ankr_getNftTransfers`](https://ankr.readme.io/reference/post_ankr-getnfttransfers) — retrieves NFT transfers info of a particular address.

### Query API

[_Query API Methods_](https://ankr.readme.io/reference/introduction#query-api) serve to request info on the ranges of blocks (max range is 100) for a full list of block metadata:

  * [`ankr_getBlockchainStats`](https://ankr.readme.io/reference/post_ankr-getblockchainstats) — retrieves blockchain statistics.
  * [`ankr_getBlocks`](https://ankr.readme.io/reference/post_ankr-getblocks) — retrieves full info of a particular block.
  * [`ankr_getLogs`](https://ankr.readme.io/reference/post_ankr-getlogs) — retrieves history data of a particular block range.
  * [`ankr_getTransactionsByHash`](https://ankr.readme.io/reference/post_ankr-gettransactionsbyhash) — retrieves the details of a transaction specified by hash.
  * [`ankr_getTransactionsByAddress`](https://ankr.readme.io/reference/post_ankr-gettransactionsbyaddress) — retrieves the details of a transaction specified by wallet address.
  * [`ankr_getInteractions`](https://ankr.readme.io/reference/post_ankr-getinteractions) — retrieves blockchains interacted with a particular wallet.
  * [`ankr_getInternalTransactionsByBlockNumber`](https://ankr.readme.io/reference/post_ankr-getinternaltransactionsbyblocknumber) — retrieves info on internal transactions by block number.
  * [`ankr_getInternalTransactionsByParentHash`](https://ankr.readme.io/reference/post_ankr-getinternaltransactionsbyparenthash) — retrieves info on internal transactions by parent hash.

### Token API

[_Token API Methods_](https://ankr.readme.io/reference/introduction#token-api) serve to request token-related data across multiple chains:

  * [`ankr_getAccountBalance`](https://ankr.readme.io/reference/post_ankr-getaccountbalance) — retrieves the balance of a particular account.
  * [`ankr_getCurrencies`](https://ankr.readme.io/reference/post_ankr-getcurrencies) — retrieves a list of currencies used on a particular blockchain.
  * [`ankr_getTokenPrice`](https://ankr.readme.io/reference/post_ankr-gettokenprice) — retrieves the price of a particular token.
  * [`ankr_getTokenHolders`](https://ankr.readme.io/reference/post_ankr-gettokenholders) — retrieves info on holders of a particular token.
  * [`ankr_getTokenHoldersCount`](https://ankr.readme.io/reference/post_ankr-gettokenholderscount) — retrieves the number of token holders.
  * [`ankr_getTokenTransfers`](https://ankr.readme.io/reference/post_ankr-gettokentransfers) — retrieves token transfers info.
